## 🇷🇺 О проекте

**ZAZAconvert** — мощный конвертер файлов . Более **30 инструментов** в одном окне — без лишних кнопок, без рекламы, без регистрации.

### 📌 Что умеет

**📄 PDF**
- PDF ↔ Word, JPG, PNG, TIFF
- Сжатие, склейка, разбивка, поворот, удаление страниц
- Извлечение текста, нумерация, водяной знак
- Установка / снятие пароля
- Конвертация в чёрно-белый (для экономии тонера)

**📊 Excel**
- Excel ↔ CSV
- Извлечение таблиц из PDF в Excel
- Защита и снятие защиты паролем

**🖼️ Фото**
- WEBP → JPG / PNG
- HEIC → JPG (фото с iPhone)
- Любое изображение → ICO (иконка приложения)

**📝 Текст и документы**
- DOCX → TXT (вытащить голый текст из Word)
- Markdown → PDF
- TXT → PDF, EPUB, CSV
- CSV ↔ JSON

**📚 Электронные книги**
- FB2 → EPUB / PDF (конвертация старых русских библиотек)
- TXT → EPUB (собери свою книгу из заметок)

**✨ Интерфейс**
- Светлая и тёмная тема
- Русский и английский язык
- Умный Drag-and-Drop прямо на карточки инструментов
- Избранные инструменты ⭐
- Сохранение настроек между запусками
